# Screenshots Directory

Place your application screenshots here with the following names:

1. `home-page.png` - Main page showing the character grid
2. `search-feature.png` - Search functionality in action
3. `house-filter.png` - House filter dropdown and results
4. `loading-state.png` - Loading spinner
5. `character-details.png` - Close-up of character cards with details

## How to Take Screenshots

1. Open the application in your browser (http://localhost:4200/)
2. Use your browser's screenshot tool or:
   - Windows: Windows Key + Shift + S
   - Mac: Command + Shift + 4
3. Save screenshots with the names listed above
4. Place them in this directory

## Screenshot Guidelines

- Use full browser window for context
- Ensure good lighting and clarity
- Show different states (loading, loaded, filtered)
- Include the browser address bar to show it's running
- Capture both the UI and some code in VS Code for code screenshots
