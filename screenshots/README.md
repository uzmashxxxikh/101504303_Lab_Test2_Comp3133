## 📝 How to Take Screenshots

### Windows:
1. Press `Windows + Shift + S`
2. Select area to capture
3. Paste into Paint or image editor
4. Save with the filename from above

### Tips:
- ✅ Include browser address bar (shows localhost:4200)
- ✅ Make sure text is readable
- ✅ Use consistent window sizes
- ✅ Capture full context (don't crop too much)
- ✅ Save as PNG or JPG format
